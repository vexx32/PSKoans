# Get-Blank

## SYNOPSIS
Returns $null

## SYNTAX

```
Get-Blank [<CommonParameters>]
```

## DESCRIPTION
Doesn't return any output.
This function exists to permit tokens such as __
to be used without quotation marks where they would be situationally appropriate.

## EXAMPLES

### EXAMPLE 1
```
Get-Blank
```

\<no output\>

### EXAMPLE 2
```
__
```

\<no output\>
