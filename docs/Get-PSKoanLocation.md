# Get-PSKoanLocation

## SYNOPSIS
Retrieves the set folder location where PSKoans' koan files are stored.

## SYNTAX

```
Get-PSKoanLocation [<CommonParameters>]
```

## DESCRIPTION
Returns the module-scoped PSKoanLocation variable.

## EXAMPLES

### EXAMPLE 1
```
Get-PSKoanLocation
```

C:\Users\Timmy\PSKoans
