# Show-Advice

## SYNOPSIS
Prints a piece of advice to the screen.

## SYNTAX

```
Show-Advice [[-Name] <String>] [<CommonParameters>]
```

## DESCRIPTION
Prints a piece of advice to the screen.

Use Get-Advice to have a random piece of advice shown on each console start

## EXAMPLES

### EXAMPLE 1
```
Get-Advice
```

Print a random piece of advice to the screen.
